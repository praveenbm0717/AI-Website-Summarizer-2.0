# 🤖 AI Website Summarizer 2.0

Summarize any website instantly using **Groq + Llama 3.3**.

This AI-powered application extracts the readable content from any webpage using BeautifulSoup and generates a concise summary using a Large Language Model (LLM). The project features a modern Gradio interface and is deployed online using Render.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/praveenbm0717/AI-Website-Summarizer-2.0.git
```

Move into the project

```bash
cd AI-Website-Summarizer-2.0
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GROQ_API_KEY=your_groq_api_key
```

Run the application

```bash
python app.py
```

---

# 🚀 Deployment

This application is deployed using **Render**.

Deployment Steps:

- Push code to GitHub
- Connect GitHub repository to Render
- Add `GROQ_API_KEY` as an Environment Variable
- Deploy

```
        ▼

        │
        ▼

(Llama 3.3)
        │
        ▼
AI Generates Summary
        │
        ▼
Display Result in Gradio
```
